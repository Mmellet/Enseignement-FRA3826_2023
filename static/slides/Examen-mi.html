Volets

Autonomie

Édition - Base à partir du Modèle
  • Restitution des mises en formes : italiques, tirets longs, espaces insécables, etc.
  • Restitution de la structure : paragraphe, note de bas de page, etc.

Modèle de l'édition 1884

Édition - + 5 Ajouts
  • Ajout de note de bas de page (x 2)
  • Insertion d'hyperliens (x 3)

Rappel Balisage

Sémantisation - + 1 donnée
  • Dans les métadonnées de votre nouvelle, ajoutez la donnée editor
  • Pour sa valeur, renseignez votre nom

Attention à la syntaxe

Enrichissement - + 1 image
  • Choisir et télécharger une image sur Wikimédia
  • Au bon endroit dans l'Arborescence du repository, importez/uploadez l'image
  • Dans votre nouvelle, insérez en markdown l'image

Attention aux chemins/paths

Suivre les instructions de l'ajout jusqu'à la pull-request

Attention à bien être sur votre branch

Attention à puller avant ajout

Un Problème : Une issue
  • Dans l'onglet Issue, sélectionnez New Issue>
  • Renseignez votre problème le plus précisément possible (comportements, manipulations, captures d'écran)
  • Assignez-moi l'issue
