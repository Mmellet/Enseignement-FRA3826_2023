Pandoc
  • Logiciel libre en ligne de commande pour la conversion de fichiers texte
  • John MacFarlane, licence GPL

Installation de Pandoc
  • Version simple
  • Mac :
  • Windows : ou
Fonctionnement en ligne de commande
  • Exemple : pandoc --standalone --bibliography Bibliographie.bib --template=Template.latex -f markdown -t latex TexteAvant.md -o TexteApres.tex
  • Traduction : cher pandoc version standalone, en utilisant ma bibliographie (qui est le fichier "Bibliographie.bib") et mon template (qui est le fichier "Template.latex"), fais la conversion, depuis (from) le Markdown vers (to) le Latex, de mon fichier de départ (TexteAvant.md) vers mon fichier de destination (out) (TexteApres.tex)
