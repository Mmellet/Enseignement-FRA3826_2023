Format et Science ouverte

Que dit-on dans le format ?
Structure d'un document numérique
  • Vue séquentielle

+ Vue stylisée (facultative)

Un format est

[s]tructure définie de données contenues sur un support magnétique ou autre, établie selon des règles qui régissent le stockage, l'affichage, la manipulation, l'impression ou la transmission de ces données.

OQLF, 2001
Critères du format
  • Pertinence
  • Économie
  • Efficacité
  • Sécurité

Composition/Interprétation de ces critères dans un format

mais un format n'est pas
  • un logiciel
  • une extension
Formats ouverts

pdf, odt, md, html

  • spécifications du format connues
  • interopérabilité favorisée
Formats fermés

doc, docx

  • spécifications du format inconnues
  • interopérabilité limitée
Formats propriétaires

doc, docx, pdf, rtf, kindle etc.

  • appartiennent à des entités privées
  • spécifications définies par un producteur commercial
  • peuvent être ouverts
Formats libres

txt, html, xml, tex, md etc.

  • appartiennent à des communautés
  • spécifications définies par les pratiques
  • se fondent sur des standards et évoluent avec les pratiques
Exemples
  • DOCX : propriétaire (Microsoft) fermé
  • PDF : propriétaire (Adobe) ouvert
  • Md : libre ouvert
