Langage pour Hypertexte

Uniform Resource Locator
  • Identification de la destination
  • Lien unidirectionnel
  • Syntaxe :
    http: ou https: protocole HTTP
    //(...) : serveur (www pour le Web)
    /(...)/ : chemin
    + extension du fichier et/ou encre avec #
