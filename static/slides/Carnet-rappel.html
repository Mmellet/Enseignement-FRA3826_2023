Attentes différentes selon chacun des modèles

Modele 1 - La Fabrique, 2 - Cupper, 3 - Book

Pour pusher en Local, il faut d'abord ajouter
  1. git add /chemin_du_fichier_modifié.extension
  2. git commit -m "commentaire pour expliquer la modfication"
  3. git push
Pour la création de pages
  • Fichier avec extension .md
Pour la création Page Crédits - Modele 1
  • création fichier dans dossier content/pages + ajout de dans fichier config.toml
Pour la création Page Crédits - Modele 2
  • création fichier dans dossier content/ + ajout de dans fichier config.toml
Pour la création Page Crédits - Modele 3
  • création fichier dans dossier content/pages/ + ajout de dans fichier content/menu/index.md
Changer le theme (css)
  • Modifier le fichier themes/roman/assets/css/style.scss
  • Modifier le fichier themes/cupper-hugo-theme/assets/css/template-style.css
  • Modifier le fichier themes/book/assets/main.scss
Modifier le footer
  • Modification de config/_default/params.toml
  • Modification à la Ligne 20 dans config.toml
  • Modification dans themes/book/layout/partials/docs/inject/footer.hmtl (vous pouvez vous inspirer du footer du cours
Changer les images - Modele 1
  • Image Logo Web : dans le dossier static/images, uploader l'image souhaitée puis dans le dossier config/_default, modifier le nom de l'image ajoutée à la ligne 11
  • Image Couverture Print : dans le dossier static/uploads, uploader l'image souhaitée puis dans le dossier config/_default, modifier le nom de l'image ajoutée à la ligne 4
Changer l'image logo - Modele 2
  • dans le dossier themes/cupper-hugo-theme/static/images uploader l'image souhaitée
  • dans le dossier themes/cupper-hugo-theme/layout, fichier index.html, modifier le nom de l'image ajoutée à la ligne 178
  • dans le dossier themes/cupper-hugo-theme/layout/partials, fichier header.html, modifier le nom de l'image ajoutée à la ligne 13
Ajout une métadonnée - Modèle 1
  • Ajout variable dans config/_default/params.toml + ajout variable en Go dans themes/roman/layout/default/list.html
  • ou Ajout variable dans les métadonnées de chaque post + ajout variable en Go dans themes/roman/layout/default/list.html

Ajout une métadonnée- Modèle 2
  • Ajout variable dans config.toml (section [params]) + ajout variable en Go dans themes/cupper-hugo-theme/layout/default/ (choix du fichier html)
  • ou Ajout variable dans les métadonnées de chaque post + ajout variable en Go dans themes/cupper-hugo-theme/layout/default/ (choix du fichier html)

Ajout une métadonnée - Modèle 3
  • Ajout variable dans config.toml (section [params]) + ajout variable en Go dans themes/book/layout/default/ (choix du fichier html)
  • ou Ajout variable dans les métadonnées de chaque post + ajout variable en Go dans themes/book/layout/default/ (choix du fichier html)
